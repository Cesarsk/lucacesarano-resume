# Resume — Luca Cesarano

Source for [Luca Cesarano](https://lucacesarano.com)'s CV. Built with [Typst](https://typst.app).

Download the latest PDF from the [Actions tab](https://github.com/Cesarsk/lucacesarano-resume/actions) — workflow artifact `lucacesarano-resume`.

## Build locally

```bash
make build    # compile main.typ → main.pdf
make watch    # live preview while editing
```

Requires `typst` (e.g. `brew install typst`).
